## 📌 Project Overview

The **AI Chatbot for College Helpdesk** is a smart virtual assistant designed to provide 24/7 support for students.  
It uses:
- **TF-IDF Vectorization**
- **Cosine Similarity**
- **Natural Language Processing (NLP)**

to understand user queries and return relevant responses.

This project is ideal for:
- B.Tech / CSE students
- AI / ML academic projects
- College helpdesk automation
